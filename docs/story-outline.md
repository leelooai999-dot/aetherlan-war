# 《埃瑟兰战记：星尘之绊》故事大纲与角色锚点

> 作用：这是角色创作与立绘解读的基准文件。以后每次根据图片生成角色时，优先回看本文件，确保角色关系、年龄、职业、气质和剧情定位不跑偏。

## 核心关系
- 塞缪尔・艾利奥特：男主角
- 伊索尔德・艾利奥特：塞缪尔的妹妹，不是恋爱向女主
- 兄妹关系是本作核心情感支柱之一，角色设计必须优先服务“亲情羁绊、共同成长、守护与传承”

## 角色锚点

### 1. 塞缪尔・艾利奥特
- 身份：男主角，伊索尔德的哥哥
- 职业路线：符文剑士 → 星辉守护者
- 战斗定位：前排输出 / 反击 / 守护
- 角色方向：肩负守护责任的王道型主角，兼具领袖感与近战压制力
- 视觉约束：设计上要体现“前线守护者、星辉之力、王庭/传承感、能保护妹妹与同伴”

### 2. 伊索尔德・艾利奥特
- 身份：塞缪尔的妹妹
- 年龄：17 岁
- 外观：容貌灵动，浅金色长发
- 天赋：继承母亲的元素魔法天赋，尤其擅长风系与治愈魔法
- 性格：活泼跳脱，好奇心极强，对星隐乡之外的世界充满向往，偶尔会因冲动闯祸，但内心善良柔软，始终坚守正义
- 初始魔兽：月溪灵鹿
  - 特征：通体莹蓝，能踏风而行
  - 能力：治愈魔法、幻术、感知周围魔法波动
  - 团队职责：团队的“移动治愈站”
- 职业路线：风语术师 → 星月灵愈使
- 战斗定位：治疗 / 控场 / 辅助输出
- 设计约束：
  - 她是妹妹系核心角色，不可误写成恋爱向女主
  - 视觉上应突出轻盈、灵动、风元素、治愈感、星月气质
  - 即使服装华丽，也不能压过“17 岁、活泼、善良、风与治愈”的角色本质
  - 武器与技能优先围绕风系、治疗、幻术、感知，而非纯爆发毁灭法术

## 立绘解读规则
以后收到角色图片时，按以下顺序判断：
1. 先看是否符合既定角色锚点，而不是只看画面酷不酷
2. 先对齐人物关系，再生成技能和人设
3. 若图片表现与大纲冲突，以大纲为准，图片只做视觉参考
4. 伊索尔德默认按“妹妹、17岁、风+治愈、月溪灵鹿搭档”解释
5. 若要新增真正的恋爱向女主，必须单独创建新角色，不得挪用伊索尔德

## 后续创作提醒
- 做角色卡时，同时写：身份、关系、职业路线、战斗定位、视觉关键词、剧情作用
- 做技能时，优先体现角色在团队中的职责，不只追求炫酷
- 做主角团时，兄妹联动技应是重点设计对象
