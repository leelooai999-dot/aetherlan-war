# PRD Outline - 埃瑟兰战记：星尘之绊

## 产品目标
打造一款可在网页端流畅运行、具备高品质视觉表现与中长线剧情体验的策略战棋游戏。

## MVP 范围
- 可进入首页
- 可查看世界观与角色介绍
- 可进入一张可玩战棋测试地图
- 支持角色移动、攻击、回合切换
- 支持一名角色与一只魔兽的共鸣技能演示

## 关键模块
1. 官网/落地页
2. 游戏主界面
3. 战棋地图系统
4. 角色数据系统
5. 技能与战斗结算
6. 剧情对话模块
7. 存档系统

## 里程碑
- M1：文档完成
- M2：前端骨架完成
- M3：战斗原型完成
- M4：首章试玩完成
- M5：内容扩展与发布
