# 《埃瑟兰战记：星尘之绊》项目提案

## 一、项目定位
- 游戏名：《埃瑟兰战记：星尘之绊》
- 类型：网页版高画质策略战棋 SRPG
- 参考方向：梦幻模拟战、火焰纹章、皇家骑士团
- 视觉方向：日式高幻想 + 韩系高精立绘 + 魔法粒子特效

## 二、核心卖点
1. 双主角叙事，兄妹双线成长
2. 人与魔兽共鸣作战
3. 100关大地图远征
4. 地形、元素、魔兽连携技系统

## 三、章节结构
- 第1章 星隐乡惊变（1-10）
- 第2章 迷雾森林与落石镇（11-20）
- 第3章 铁脊王国阴谋（21-30）
- 第4章 月影精灵森林（31-40）
- 第5章 霜雪王国极寒战线（41-50）
- 第6章 澜汐王国遗迹群（51-60）
- 第7章 遗忘神殿远征（61-70）
- 第8章 暗影裂隙外围战争（71-80）
- 第9章 裂隙深处真相（81-90）
- 第10章 星尘之绊终章（91-100）

## 四、核心玩法
### 1. 共鸣系统
- 角色与专属魔兽相邻时触发共鸣加成
- 可释放共鸣技

### 2. 元素地形
- 火、冰、风、水、暗、光六类地形影响移动与战斗

### 3. 魔兽契约养成
- 升级、进化、觉醒、外观变化

### 4. 关卡目标多样化
- 歼灭、护送、守城、限时突围、机关解谜、多线分兵

### 5. 羁绊系统
- 剧情对话、角色协同、组合奥义

## 五、主角团职业定位
### 塞缪尔
- 符文剑士 → 星辉守护者
- 前排输出 / 反击 / 守护

### 伊索尔德
- 风语术师 → 星月灵愈使
- 治疗 / 控场 / 辅助输出

### 雷欧纳德
- 铁壁骑士 → 大地战将
- 坦克 / 冲锋 / 破阵

### 莉奥拉
- 月影游侠 → 森灵追猎者
- 高速刺客 / 侦查 / 弱点打击

## 六、技术方案
- 前端：Next.js
- 渲染：PixiJS 或 Phaser
- UI：React + Tailwind
- 后端：Node.js / NestJS
- 数据：PostgreSQL
- 部署：Vercel + 独立游戏服务

## 七、美术方向
- 主视觉：星辉、符文、精灵森林、暗影裂隙
- 主色调：蓝银、月白、翡翠绿、暗紫黑
- UI风格：金属纹章 + 水晶魔法面板
- Logo方向：古典幻想字体 + 星尘拖尾 + 裂纹符文感

## 八、商业化建议
- 前20关免费
- 后续章节解锁 / 季票
- 皮肤、魔兽外观、特效包付费
- 避免重氪数值碾压
- 剧情DLC、挑战塔、限时远征

## 九、执行顺序
1. 完整策划案
2. 开发 PRD
3. 首章 1-10 关详细关卡设计
4. 美术规范与官网落地页
5. 可玩原型
6. 完整内容生产与上线
